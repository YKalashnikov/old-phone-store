# Old Phones Store (Node.js, materialize)

#### My Stack

- Client
  - Materialize
- Database
  - MongoDB
- Marketing Technology
  - SendGrid
  - nodemailer
- Server/API
  - bcryptjs
  - compression
  - connect-flash
  - csurf
  - express
  - express-handlebars
  - express-session
  - helmet
  - mongoose
  - multer
  - nodemon
  
 ```shell
$ npm install
$ npm run dev
$ go to http://localhost:3000
```
